## 	:toolbox: Getting Started

<!-- Prerequisites -->

This project uses poetry as package manager

You can follow this link to set-up https://python-poetry.org/docs/#installing-with-the-official-installer

<!-- Installation -->
### :gear: Installation

Install my-project with poetry

```bash
   poetry install
```
   

<!-- Run Locally -->
### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/akirawuc/skitties.git
```

Go to the project directory

```bash
  cd skitties
```

Install dependencies

```bash
  poetry install
```

Start the server

```bash
  poetry run python3 fc_auth.py
```
